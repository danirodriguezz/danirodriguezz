* 💼 [**DevJobs**](https://github.com/danirodriguezz/DevJobs): Portal de empleo funcional y sencillo desarrollado en el backend con **Laravel**.
* 🎟️ [**LiveSportsPromo**](https://github.com/DACD-Daniel-y-Pablo/LiveSportsPromo): Aplicación en **Java** que combina datos de eventos deportivos en vivo y redes sociales con merchandising dinámico.
* 🕸️ [**Algoritmos de Búsqueda en Grafos**](https://github.com/danirodriguezz/Algoritmos-de-busquedad-en-grafos): Implementación en **Python** de algoritmos de búsqueda aplicados a sistemas inteligentes.
